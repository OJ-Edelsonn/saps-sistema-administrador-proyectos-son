# Pendientes y notas de entorno

No quedan pendientes criticos de codigo detectados por compilacion Maven.

Para la prueba funcional manual:

1. Abrir XAMPP.
2. Activar Apache y MySQL.
3. Crear o importar la base `saps_db`.
4. Ejecutar la aplicacion desde NetBeans o Spring Tools.

Limitaciones del entorno usado para generar el proyecto:

- No habia Maven instalado en PATH; se uso Maven portable temporal para validar compilacion.
- No habia `gh` instalado, por lo que la publicacion a GitHub se hara con `git`.
- No habia LibreOffice/`soffice` instalado, por lo que no se pudo renderizar visualmente `docs/Informe_SAPS.docx`.
