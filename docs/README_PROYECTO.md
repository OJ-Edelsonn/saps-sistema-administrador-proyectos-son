# README del proyecto S.A.P.S

S.A.P.S administra proyectos academicos integradores mediante una arquitectura MVC por capas. El sistema integra autenticacion, roles, CRUDs principales, dashboard, reportes y datos de prueba para sustentacion.

## Justificacion tecnologica

Spring Boot simplifica la configuracion del backend, Thymeleaf permite vistas server-side integradas con MVC, Spring Data JPA reduce codigo de persistencia y MySQL es adecuado para un modelo relacional academico.
