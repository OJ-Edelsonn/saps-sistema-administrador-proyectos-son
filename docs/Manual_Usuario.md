# Manual de usuario

1. Ingrese a `http://localhost:8080`.
2. Autentiquese con un usuario de prueba.
3. Use el menu lateral para navegar por los modulos.
4. En cada modulo puede listar, buscar, crear, editar y eliminar registros.
5. En dashboard revise indicadores generales.
6. En reportes consulte resumenes imprimibles del sistema.
